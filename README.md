# Arduino Robotics and Automation Projects

A collection of Arduino based robotics, automation, embedded systems, and IoT projects developed to explore practical engineering concepts through hands on experimentation.

This repository serves as a growing archive of Arduino projects covering autonomous robots, sensor based systems, environmental monitoring, and intelligent control applications. Each project is designed to demonstrate hardware integration, embedded programming, and real world problem solving using widely available electronic components.

## Projects Included

* Obstacle Avoiding Robot
* Bluetooth Controlled Robot
* Line Follower Robot
* Fire Fighting Robot
* Light Following Robot
* Gesture Controlled Robot
* Rain Sensing Robot
* Robotic Arm Control
* Smart Rescue Robot
* AI Water Quality Predictor
* Smart Parking Assistant

## Repository Structure

Each project is contained in its own directory and includes the Arduino source code required to build and test the project. Most examples are self contained and can be adapted for different hardware configurations or extended for more advanced robotics and automation applications.

## Purpose

This repository is intended as a reference for students, hobbyists, and robotics enthusiasts interested in learning Arduino programming, embedded systems, sensor integration, and autonomous robotics through practical projects.
